# Начало работы

Игровой движок **eocc** находится на ранней стадии разработки, но с самого начала в нём заложена ключевая особенность: **кроссплатформенность**.

На данный момент поддерживаются:
- **Linux** (разработка ведётся на Ubuntu 24.04),
- **Windows**,
- **macOS** (проверено на устройствах с процессорами Intel; поддержка Apple Silicon (M1/M2/M3) пока не тестировалась).

Это стало возможным благодаря использованию:
- **CMake** — для кроссплатформенной сборки,
- **GLFW** — для создания окна и обработки ввода,
- **OpenGL Core Profile** — для отрисовки (через загрузчик функций **GLAD**).

> Движок написан на **C++20** и не содержит платформо-специфичного кода.

## Сборка

Убедитесь, что установлены следующие зависимости:
- Компилятор с поддержкой C++20 (например, **Clang** или **GCC ≥ 10**),
- **CMake ≥ 3.28**,
- **Ninja** (рекомендуется),
- Заголовки OpenGL (на Ubuntu: `libgl1-mesa-dev` и `xorg-dev`).

Затем выполните:

```bash
cmake -B build -G Ninja
cmake --build build
./build/eocc_demo
```
